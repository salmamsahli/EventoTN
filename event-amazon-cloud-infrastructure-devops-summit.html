Amazon Cloud Infrastructure & DevOps Summit

Register now 📅 Add to calendar

About the event

The Amazon Cloud Infrastructure & DevOps Summit is a globally accessible virtual event connecting cloud engineers, DevOps practitioners, and system architects with the latest innovations in AWS-powered infrastructure.

The summit covers the full spectrum of modern cloud engineering — from containerized microservices and serverless architectures to CI/CD pipelines, infrastructure as code, and site reliability engineering best practices.

Participants will leave with actionable knowledge on how to design, deploy, and maintain robust cloud environments at scale, leveraging the breadth of Amazon Web Services to drive operational excellence.

What to expect

  • Keynotes from AWS engineers and cloud infrastructure leaders
  • Sessions on serverless, containers, and Kubernetes on AWS
  • Workshops on CI/CD pipelines and infrastructure as code
  • Discussions on distributed systems and site reliability engineering
  • Live demos of AWS DevOps and monitoring tools
  • Networking with global cloud and DevOps professionals
